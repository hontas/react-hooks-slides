# What are hooks?

- Allow access to features previously requiring a class
- A Remedy for HOC “wrapper hell”
- Hooks allow you to reuse stateful logic without changing your component hierarchy.

---

# Why are they useful

- Solve a wide variety of seemingly unconnected problems
- Reuse stateful logic between components
- Test stateful logic independently

https://reactjs.org/docs/hooks-intro.html#motivation
